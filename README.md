# Maze Game

A text-based maze game in Python where you navigate a randomly generated grid to reach the goal.

---

## Gameplay

Enter your name, then navigate through a 20×10 maze using `W`, `A`, `S`, `D`. The maze is randomly generated each run with walls (`#`), open paths, a player (`P`), and a goal (`X`). Reach `X` to win. Your progress percentage and step count are shown after each move.

**Controls:** `W` up · `S` down · `A` left · `D` right

---

## Features

- Randomly generated maze every run for replayability
- Progress tracker shown as a percentage toward the goal
- Step counter at the end of each game
- Wall and open path stats displayed at the start
- Input validation with helpful error messages

---

## How to Run

**Requirements:** Python 3

```bash
python MazeGame.py
```

---

## Built With

- **Python** — all game logic
- **random** — procedural maze generation
